# 08 — CSS Grid (2D)

## Objective
Create a photo gallery where one item spans multiple columns.

## Grid vs Flexbox
- **Flexbox** is best for **1D layout** (a row *or* a column).
- **Grid** is best for **2D layout** (rows **and** columns at the same time).

## Challenge
In `style.css`:
1. Make `.gallery` a 3-column grid
2. Add a gap between items
3. Make the first tile span **2 columns**

## Done When
- You have exactly 3 columns.
- Tile 1 clearly spans across two columns.
