# 10 — Shadows and State

## Objective
Create a card that “lifts” when hovered.

## Concepts
- `box-shadow`: gives depth
- `transition`: smooths changes over time
- `:hover`: a state pseudo-class triggered by pointing devices

## Challenge
In `style.css`:
1. Add a `transition: 0.3s ease` to the card
2. On `.card:hover`, apply a stronger `box-shadow` and a slight upward movement

## Done When
- Hovering feels smooth (not instant).
- The card visibly lifts with a stronger shadow.
