# 11 — Media Query Mastery

## Objective
Change the entire layout for mobile devices.

## Why Media Queries Matter
Media queries let you apply different CSS based on device characteristics (most commonly width). They’re often used to:
- stack columns on small screens
- change font sizes or spacing for readability

## Challenge
In `style.css`, write a media query for screens under `600px` that:
- changes `.layout` from `flex-direction: row` to `flex-direction: column`

## Done When
- Desktop: sidebar + content are side-by-side.
- Mobile (<600px): sidebar stacks above content.
