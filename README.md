# CSS Mastery Drills

A hands-on CSS training project built as a set of small, focused drills plus a capstone.

## How to Use This Repo
- Work in order: `01/` → `11/`, then `12-capstone-quill-project/`.
- Each drill folder includes:
  - `README.md` (the lesson + the challenge)
  - `index.html` (starter markup)
  - `style.css` (your TODOs)
- Open each `index.html` (or `starter.html` for the capstone) in your browser and edit the corresponding `style.css`.

## Project Map
- `GLOSSARY.md` — shared definitions used across drills
- `01/` — Specificity Sports Table
- `02/` — Box-Sizing Predictability
- `03/` — Display Logic Buttons
- `04/` — The REM Trick
- `05/` — Dynamic Calc Hero
- `06/` — Responsive Containers
- `07/` — Nested Flex Navbar
- `08/` — CSS Grid (2D)
- `09/` — Precision Positioning
- `10/` — Shadows and State
- `11/` — Media Query Mastery
- `12-capstone-quill-project/` — Capstone landing page roadmap + starter

## Suggested Workflow
1. Read the folder `README.md`
2. Make the minimal CSS changes to complete the “Done When” checklist
3. Don’t move on until the current drill looks correct in the browser
